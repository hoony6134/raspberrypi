# Raspberry Pi
## 라즈베리파이 기본 명령어 등
* ID: pi / PW: pi
* 온도 확인: cat /sys/class/thermal/thermal_zone0/temp
* IP주소 확인: ifconfig
* SSH 통한 접속: ssh pi@ip주소
* 라즈베리파이 설정 열기: sudo raspi-config
